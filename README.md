# llm-postman-new-version-converter
This project is a Streamlit-based web application that converts Postman collections from v2.1.0 to modern v2.2.0 script format using an LLM-powered backend.

It allows users to:

Upload a ZIP file of Postman collections
Automatically convert legacy scripts (pre-request & test scripts)
Validate collections against Postman schema
Download the converted collections as a ZIP
